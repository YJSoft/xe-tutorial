## 자바스크립트와 jQuery 라이브러리 사용

XE는 자바스크립트를 쉽게 사용할 수 있도록 jQuery라는 자바스크립트 라이브러리를 사용하고 있습니다. jQuery를 사용하면 스킨 제작자가 자바스크립트 전문가가 아닌 경우에도 자바스크립트를 쉽게 다룰 수 있습니다.

이 절에서는 XE 스킨을 제작할 때 HTML 문서에 자바스크립트를 포함시키는 방법을 설명합니다.
