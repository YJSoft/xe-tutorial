## 레이아웃 스킨의 위치 확인

*/xe/*라는 사용자 정의 디렉터리에 XE core를 설치했다면 레이아웃 스킨 디렉터리의 위치는 다음과 같습니다.

```
/xe/layouts/
```

> 레이아웃 스킨을 로컬 PC에서 수정할 때는 수정한 내용을 웹 사이트의 레이아웃 스킨 디렉터리에 반영해야 합니다.
