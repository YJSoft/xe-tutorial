## 사용자 정의 레이아웃 확인

*info.xml*을 올바르게 작성했는지 확인하는 방법은 다음과 같습니다.

1. XE 관리자 페이지에서 *고급 > 설치된 레이아웃*을 엽니다.
2. 테스트 레이아웃이 제대로 표시되는지 확인합니다.
