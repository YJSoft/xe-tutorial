## XE 템플릿 문법

XE 템플릿 문법은 서버에서 PHP 문법으로 해석되며 DB로부터 원하는 정보를 뽑아내어 사용자 화면에 출력합니다. XE 템플릿 문법을 사용하면 모듈 또는 위젯의 기능을 허용된 범위 안에서 확장하거나 축소할 수 있습니다.
